# Weather_App_Mini_Project

# 🌦️ Real-Time Weather Dashboard
A modern, interactive weather dashboard built using **HTML, CSS, and JavaScript**, featuring real-time weather data, dynamic UI effects, and smooth animations.
____________________________________________________________________________________________________________________________________________________________________

## 🚀 Live Features

* 🔍 **Search Any Location**
  Get real-time weather data for any city worldwide.

* 📍 **Auto Location Detection**
  Detects user location using browser Geolocation API.

* 🌈 **Dynamic Backgrounds**
  UI changes based on weather conditions (sunny, cloudy, rain, night).

* ☀️ **Animated Weather Effects**

  * Glowing animated sun
  * Floating clouds
  * Real rain particle animation

* 📊 **6-Day Forecast**
  Displays upcoming weather conditions with icons and temperatures.

* 💾 **Persistent Search**
  Saves last searched city using `localStorage`.

* ⏳ **Loading State**
  Smooth loader while fetching API data.

* ❌ **Error Handling**
  Handles invalid input, API failures, and location denial gracefully.
  __________________________________________________________________________________________________________________________________________________________________
  
## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Glassmorphism, Animations, Gradients)
* **JavaScript (ES6+)**
* **WeatherAPI** (for real-time data)
* **Canvas API** (for rain animation)
* **Geolocation API**

## ⚙️ Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/Manasa-Sarva/Weather_App_Mini_Project.git
```

2. Navigate to project folder:

```bash
cd Weather_App_Mini_Project
```

3. Open `index.html` in your browser

## 🔑 API Configuration

This project uses **WeatherAPI**.

1. Get your free API key from:
   👉 https://www.weatherapi.com/

2. Replace API key in `script.js`:

```javascript
const API_KEY = "YOUR_API_KEY";
```
